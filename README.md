# Fiesto – AI-Powered Dinner Planning App

Fiesto is a University group project developed as part of a software engineering course.

Fiesto is a mobile app that helps groups plan dinners together. Users can create events, invite friends, generate AI-powered recipes based on dietary preferences and allergies, track costs, and manage upcoming dinners.

## My Contribution

I was responsible for the UI/UX design and Flutter frontend implementation, including:

* Designing the user interface and user flows
* Building responsive screens and reusable UI components in Flutter
* Implementing navigation and app theming
* Creating a consistent user experience across the application

## Key Features

* AI-generated recipes based on group preferences and allergies
* Dinner event creation and guest invitations
* Calendar view for upcoming and previous events
* Favorite recipes
* Cost sharing and payment reminders
* Push notifications

## Technologies

* Flutter & Dart
* Supabase
* Firebase Cloud Messaging
* OpenAI API
* Google Places API
* Git & GitHub

## Project Information

This project was developed collaboratively in a university team environment. The source code is maintained in a private repository.
